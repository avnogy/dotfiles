# Dotfiles

My dotfiles, managed with [chezmoi](https://www.chezmoi.io/).

## Stuff i currently use and like

- [awesomewm](https://awesomewm.org/) - window manager.
- [yshui/picom](https://github.com/yshui/picom) - a compositor for X11.
- [alacritty](https://alacritty.org/) - terminal emulator.
- [pythops/impala](https://github.com/pythops/impala) - tui for wifi,
- [pythops/bluetui](https://github.com/pythops/bluetui) - tui for bluetooth.
- [pulsemixer](https://salsa.debian.org/debian/pulsemixer) - tui mixer for PulseAudio.
- [zoxide](https://github.com/ajeetdsouza/zoxide) - fast cd.

I use Debian btw.
